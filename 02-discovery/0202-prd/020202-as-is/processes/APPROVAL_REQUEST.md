## ¿Cómo aprobar?

Responda en este chat con un bloque:

```APPROVE-LIST
+ {slug-o-nombre} -> APPROVE         [opcional: nuevo-nombre | merge-with:{slug}]
+ {slug-o-nombre} -> REJECT          [motivo]
+ {slug-o-nombre} -> DEFER           [motivo]
```

También puede aprobar todo:

`APPROVE-LIST: APPROVE ALL`
