# kamleonapp-facturacionOdoo
Análisis funcional del módulo Finanzas de Odoo para KameleonLabs

## Paquete de implementación: Módulo de facturación Kameleon App

En la carpeta **`modulo-facturacion-kameleon/`** se mantiene un paquete con la **misma estructura** que este repositorio, usado como **guía para implementar** los requisitos del módulo de facturación de Kameleon App. Incluye:

- **02-discovery**: contexto, TO-BE (factura proforma, factura final), alcance, 2 épicas (MF-001, MF-002) y 9 historias de usuario.
- **01-lead**, **03-poc**: brief y POC (esqueleto).

La estructura y los requisitos del paquete de referencia (27 épicas, 133 US) sirven de guía; en `modulo-facturacion-kameleon` solo se desarrollan los requisitos de facturación (trazabilidad con EP-010 y EP-022).
