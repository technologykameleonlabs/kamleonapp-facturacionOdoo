# Índice TO-BE — Módulo de facturación

Procesos TO-BE del módulo de facturación Kameleon App. Guía: TO-BE-010 y TO-BE-022 del paquete de referencia.

| ID | Proceso | Referencia guía |
|----|---------|-----------------|
| TO-BE-MF-01 | Activación de proyecto (sin pago obligatorio) | TO-BE-010 |
| TO-BE-MF-02 | Factura de cierre / liquidación y registro de pago | TO-BE-022 |

**Dependencia**: TO-BE-MF-02 requiere el evento de cierre del proyecto (última entrega aceptada o "listo para cerrar"); TO-BE-MF-01 habilita proyecto activo (por contrato firmado o manual) y opcional reserva de fecha y factura de anticipo.
