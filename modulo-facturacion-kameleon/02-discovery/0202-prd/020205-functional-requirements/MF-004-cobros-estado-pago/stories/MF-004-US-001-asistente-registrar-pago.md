# MF-004-US-001 — Asistente Registrar pago: fecha, importe, método de pago, referencia

**Epic**: MF-004 — Cobros y estado de pago

**Como** usuario con permiso de registrar cobros, **quiero** abrir un asistente para registrar un pago indicando fecha, importe, método de pago y referencia (opcional), **para** poder aplicarlo luego a una o varias facturas.

**Criterios de aceptación**: Asistente o formulario "Registrar pago"; campos fecha, importe, método de pago, referencia; guarda la entidad Pago/Cobro; siguiente paso es aplicar a factura(s).

**Prioridad**: Alta
