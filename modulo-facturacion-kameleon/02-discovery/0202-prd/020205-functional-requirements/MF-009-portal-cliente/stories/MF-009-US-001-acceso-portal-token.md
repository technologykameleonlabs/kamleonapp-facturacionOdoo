# MF-009-US-001 — Acceso al portal de facturación con token mágico o login cliente

**Epic**: MF-009 — Portal del cliente

**Como** cliente, **quiero** acceder a un área donde ver mis facturas mediante un enlace con token o con mi usuario de cliente, **para** no necesitar acceso al back-office de Kameleon.

**Criterios de aceptación**: URL de portal (o enlace con token); autenticación por token mágico o por credenciales cliente; solo se muestran facturas del cliente identificado.

**Prioridad**: Alta
