# NFR — Módulo de facturación

Requisitos no funcionales aplicables al módulo de facturación. Completar según necesidad (seguridad, rendimiento, cumplimiento, usabilidad).

**Referencia**: En el paquete de referencia, NFR en `020206-non-functional-requirements` (compliance, security, performance, etc.). Para facturación son relevantes: confidencialidad de datos de facturación y pagos, trazabilidad y auditoría.
